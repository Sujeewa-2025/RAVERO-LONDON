# RAVERO LONDON Website

Static premium landing page for GitHub Pages hosting.

## Files
- `index.html` — main website page
- `styles.css` — all website styling
- `script.js` — small interaction/year script
- `assets/` — logo and watch images

## GitHub Pages upload steps
1. Create a new GitHub repository, for example `ravero-london`.
2. Upload all files and folders from this package to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. GitHub will generate your live website URL.

## Replacing images
Keep the same filenames in `/assets/` if you want to replace the images without changing the code.
